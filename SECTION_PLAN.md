# SECTION PLAN — 8-SLIDE IMMERSIVE PORTFOLIO STRUCTURE

## Execution Rule

Build one section at a time.

Do not build the full website in one pass.

Required workflow:

1. Global environment setup only.
2. Build Slide 1 only.
3. Review and refine Slide 1.
4. Build Slide 1 to Slide 2 transition only.
5. Build Slide 2 only.
6. Continue section by section.

Every section must follow the motion rules in `MOTION_RULES.md`.

Most important rule:

No basic transitions.
No static important elements.
Every important UI element must feel alive.
Build every slide as real interactive website sections using real components, animation layers, scroll-driven transitions, and responsive UI.
Do not use reference images as static backgrounds.
