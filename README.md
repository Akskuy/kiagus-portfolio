## 1. Project Direction

This is not a normal portfolio website.

This project is a premium cinematic portfolio built as a one-viewport interactive journey.

The website should feel like:

* a cinematic data system
* a professional AI/Data Science identity
* an interactive technical portfolio
* a premium custom-built experience
* a system-oriented product preview

The website must not feel like:

* a basic student CV page
* a normal vertical scrolling portfolio
* a template landing page
* a static screenshot gallery
* a PowerPoint deck
* a Figma prototype
* a fake expert site

Target direction:

**80% professional portfolio**
**20% cinematic/game-inspired interaction**

---

## 2. Main Experience Rule

The portfolio must use a one-viewport cinematic journey.

Correct behavior:

User scrolls
→ viewport appears pinned
→ objects transform inside the same screen
→ next scene assembles inside the same screen

Wrong behavior:

User scrolls
→ page moves downward
→ next section appears below
→ it feels like PDF / Figma / normal website scrolling

The wrong behavior is forbidden.
